# The bar left of the Neobrains IDE for choosing between file explorer and git and what not

## Install

### With Lazy

```lua
{
  "SCRIPTERBLOX/neobrains-left-bar-select.nvim",
  name = "neobrains-left-bar-select"
}
```

## Documentation

### Config

```lua
buttons = <table<button>> -- The buttons/elements to display on the sidebar
```

```lua
width = <integer> -- How wide the sidebar is
```
